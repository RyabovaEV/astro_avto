# Импорт расписаний из TSV

## Назначение
Папка хранит входные TSV-файлы, из которых генерируются:
- `src/data/schedule/suburban.ts`
- `src/data/schedule/city.ts`
- `src/data/schedule/suburbanComments.ts`
- `src/data/schedule/cityComments.ts` (если есть `citycomments_*.tsv`)

## Поддерживаемые файлы
- `suburban_*.tsv` — пригородные маршруты
- `city_*.tsv` — городские маршруты
- `subcomments_*.tsv` — комментарии к пригородным маршрутам
- `citycomments_*.tsv` — комментарии к городским маршрутам (опционально)

Скрипт берет последний файл каждого типа по дате изменения.

## Запуск импорта
```bash
npm run import:schedule
```

## Правила нормализации
- `kind`: `TRUE -> circular`, `FALSE -> linear`
- `dep_days/arr_days`: в импорте `,` заменяется на `;`
- `route_number`: сохраняется как есть (включая `*`, `№`, буквенные суффиксы)
- `route_name`: сохраняется, лишние пробелы и переводы строк внутри ячейки нормализуются
- время приводится к `HH:MM`

## Политика хранения
Храним последние 4 файла каждого типа. При добавлении новых файлов заменяем самые старые.
